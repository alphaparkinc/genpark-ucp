# GenPark Appediet Health Expert Skill Integration

This repository contains the reverse-engineered logic for **Appediet.com**, integrated as a GenPark Skill.

## Features
- **AI Food Analysis**: Multi-modal vision processing for nutritional identification.
- **Personalized Health Expert (AppeChat)**: Conversational nutrition and hydration advice.
- **Smart Recipe Recommendation**: Data-driven meal suggestions based on health goals.
- **Diet Evaluation**: Real-time analysis of logged daily intakes.

## Repository Structure
- `skills/genpark-health-expert/SKILL.md`: Detailed documentation of the health & nutrition agent.
- `skills/genpark-health-expert/scripts/appediet_engine.py`: Core logic for food identification and calorie mapping.

## Integration with GenPark
The Health Expert engine provides GenPark users with a personal "Health Assistant" that integrates food scanning and macro-tracking directly into their daily AI flow.
