# GenPark RedNote Logic Integration

This repository contains the reverse-engineered logic for **Xiaohongshu (Little Red Book)**, integrated as a GenPark Skill.

## Features
- **Visual "Notes" Generation**: High-aesthetic visual storytelling combined with informative, "seed-planting" (种草) micro-blog captions.
- **KOC-style Authenticity Engine**: Logic to generate authentic-feeling user reviews and experiences.
- **Keyword-to-Tag Mapping**: Automatically extracts trending lifestyle tags (e.g., #OOTD, #MyHomeVibe).
- **"Seed Planting" (种草) Analysis**: Evaluates product mentions for potential virality and conversion.
- **Interactive Comment Simulation**: Mimics XHS community engagement dynamics.

## Repository Structure
- `skills/genpark-rednote-logic/SKILL.md`: Detailed documentation of the visual discovery and "Note" agent.
- `skills/genpark-rednote-logic/scripts/rednote_engine.py`: Core logic for content synthesis and discovery matching.

## Integration with GenPark
RedNote Logic provides a high-aesthetic, community-driven discovery layer to the GenPark workspace, enabling users to "seed plant" (种草) products and share lifestyle "notes" in a non-linear, visual flow.
