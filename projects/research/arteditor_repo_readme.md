# GenPark ArtEditor Skill Integration

This repository contains the advanced design logic for **ArtEditor**, integrated as a GenPark Skill.

## Features
- **Multi-Agent Orchestration**: Research, Visualize, Edit, and Synthesize.
- **Layer-aware Image Editing**: 'Touch Edit' philosophy (precise element swapping).
- **Brand System Consistency**: Unified color and voice across campaign assets.
- **Real-time Design Reference**: Live web search integration for trend curation.

## Repository Structure
- `skills/genpark-arteditor-engine/SKILL.md`: Detailed documentation of the ArtEditor workflow.
- `skills/genpark-arteditor-engine/scripts/arteditor_engine.py`: Core logic for intent analysis and multi-agent coordination.

## Integration with GenPark
The ArtEditor Engine is designed to work within the GenPark "ChatCanvas" system, providing users with editable design layers rather than static AI-generated images.
