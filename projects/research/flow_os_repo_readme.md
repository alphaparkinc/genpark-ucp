# GenPark FlowOS Skill Integration

This repository contains the reverse-engineered logic for **Flowith.io**, integrated as a GenPark Skill.

## Features
- **Infinite Canvas Orchestration**: Visualizes multi-step AI reasoning as nodes on a non-linear playground.
- **Seed Decomposition**: Automatically breaks down complex materials/prompts into atomic "Seeds" (core concepts).
- **Knowledge Garden Integration**: Unified context system that cultivates long-term memory and retrieval.
- **Agent Neo Core**: High-throughput execution agent designed for million-token context handling.
- **Dynamic Data Widgets**: Real-time integration of financial news and stock tickers.

## Repository Structure
- `skills/genpark-flow-os/SKILL.md`: Detailed documentation of the Flowith-style workflow.
- `skills/genpark-flow-os/scripts/flow_os_engine.py`: Core logic for task decomposition and node-based execution.

## Integration with GenPark
FlowOS provides a non-linear workflow for GenPark users, allowing them to visualize complex agentic tasks on an infinite canvas rather than a single chat thread.
