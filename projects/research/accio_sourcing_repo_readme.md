# GenPark Accio Sourcing & Procurement Skill Integration

This repository contains the reverse-engineered logic for **Accio.com (Alibaba AI Business Agent)**, integrated as a GenPark Skill.

## Features
- **AI B2B Sourcing Engine**: Natural language procurement intent to verified supplier discovery.
- **Real-time Pricing & Market Insights**: Continuous updates on supplier listings and global trade trends.
- **AI Knowledge Wiki Integration**: Manufacturing standards, shipping logistics, and compliance requirements.
- **Automated Trend Analysis**: Identifies high-demand product niches and sourcing opportunities.
- **Profit Margin Calculation**: Automated tools to estimate landed costs and import duties.

## Repository Structure
- `skills/genpark-accio-sourcing/SKILL.md`: Detailed documentation of the sourcing and procurement agent.
- `skills/genpark-accio-sourcing/scripts/accio_engine.py`: Core logic for B2B intent parsing and supplier ranking.

## Integration with GenPark
Accio Sourcing provides a professional B2B layer to the GenPark workspace, enabling users to move from "Idea" to "Profitable Sourcing" by automating the search for manufacturers and the calculation of business margins.
