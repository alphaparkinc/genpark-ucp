# GenPark Lemon8 Lifestyle Sharing Skill Integration

This repository contains the reverse-engineered logic for **Lemon8-app.com**, integrated as a GenPark Skill.

## Features
- **Aesthetic Post Generation**: Automates high-quality visual content creation with informative, blog-style captions.
- **Interest-based Curation (Discovery Engine)**: Mimics the "For You" algorithm for lifestyle theme categorization.
- **Smart Template System**: Standardized visual layouts for "Outfit of the Day", "Healthy Recipes", and "Travel Guides".
- **Community Engagement Simulation**: Automates authentic-feeling social interactions and appreciation patterns.
- **SEO-optimized Tagging**: Recommends viral hashtags based on real-time lifestyle trends.

## Repository Structure
- `skills/genpark-lemon-lifestyle/SKILL.md`: Detailed documentation of the lifestyle sharing agent.
- `skills/genpark-lemon-lifestyle/scripts/lemon_engine.py`: Core logic for aesthetic curation and micro-blogging.

## Integration with GenPark
The Lemon8 Lifestyle engine enables GenPark users to curate, create, and share high-aesthetic content within a community-focused interest engine, blending visual inspiration with helpful advice.
