# GenPark BrandTester Discovery & Review Skill Integration

This repository contains the reverse-engineered logic for **Thingtesting.com**, integrated as a GenPark Skill.

## Features
- **Brand Discovery Engine**: Automated scanning and curation of emerging DTC (Direct-To-Consumer) brands.
- **Honest Review Synthesis**: Multi-source sentiment aggregation (Reddit, Trustpilot, Thingtesting) into a unified "GenPark Review".
- **Hype vs. Value Analysis**: Comparing brand claims against community feedback to identify "Hype Markers".
- **Comparison & Alternatives Engine**: AI-driven logic to find and compare better-rated competitors.
- **Community Engagement Logic**: Points/level system (Thingtester Style) to reward high-quality contributions.

## Repository Structure
- `skills/genpark-brand-tester/SKILL.md`: Detailed documentation of the brand discovery and review agent.
- `skills/genpark-brand-tester/scripts/brand_tester_engine.py`: Core logic for brand scanning and sentiment synthesis.

## Integration with GenPark
BrandTester enables GenPark users to move beyond marketing hype, providing honest, community-driven insights and discovering new products that offer real value.
