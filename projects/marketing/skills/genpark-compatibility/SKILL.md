# GenPark Compatibility-Judge Skill (Ecosystem Integrity)

Analyzes the physical and protocol compatibility of a new product with the user's existing "GenPark Asset Database" (home network, physical dimensions, software ecosystem) before purchase.

## Commands

### `genpark-judge check <product_url>`
Performs a cross-reference between the product's technical specs and the user's current environment (e.g., "Will this RAM fit the motherboard?" or "Is this bulb compatible with my Zigbee hub?").

### `genpark-judge find-fit <product_category>`
Recommends only products that are 100% compatible with the user's current tech stack, preventing "return fatigue."

## Guidelines
- Priority: Zero-friction integration and zero-return policy.
- Focus: Smart home, PC hardware, and complex electronics.
- Tone: Clinical, precise, and authoritative.
