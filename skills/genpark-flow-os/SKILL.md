# Skill: genpark-flow-os
# Description: Reverse-engineered Flowith.io agentic workspace. Implements a multi-agent canvas-based flow for knowledge synthesis, task decomposition (Seeds), and autonomous execution.

## Core Capabilities
1. **Infinite Canvas Orchestration**: Visualizes multi-step AI reasoning as nodes on a non-linear playground.
2. **Seed Decomposition**: Automatically breaks down complex materials/prompts into atomic "Seeds" (core concepts) for precise AI processing.
3. **Knowledge Garden Integration**: A unified context system that cultivates long-term memory and cross-project knowledge retrieval.
4. **Agent Neo Core**: High-throughput execution agent designed for million-token context handling and continuous task loops.
5. **Dynamic Data Widgets**: Real-time integration of financial news, stock tickers, and economic calendars into the flow.

## Workflow Logic
- **Stage 1 (Plan)**: Agent Neo analyzes the prompt and spawns a "Thinking Canvas".
- **Stage 2 (Decompose)**: Complex inputs are shattered into "Seeds" stored in the Knowledge Garden.
- **Stage 3 (Execute)**: Parallel sub-agents (Researchers, Analysts) process Seeds across the canvas.
- **Stage 4 (Synthesize)**: Merges disparate nodes into a cohesive final output (Market Pulse, Report, etc.).
