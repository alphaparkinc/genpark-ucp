# Skill: genpark-agentrun-engine
# Description: Reverse-engineered AgentRun always-on AI workforce system. Orchestrates persistent, proactive AI agents that execute end-to-end workflows on dedicated cloud VMs.

## Core Capabilities
1. **Always-On Persistence**: Agents run 24/7 on dedicated virtual environments, continuing tasks even when the user is offline.
2. **Proactive Monitoring & Action**: Monitors metrics (uptime, pricing, inventory) and executes pre-defined "next steps" autonomously when triggers occur.
3. **Collective Intelligence Learning**: Agents self-evolve by learning from successful workflow patterns and shared business solutions within the network.
4. **End-to-End Tool Usage**: Unlike chatbots, these agents interact directly with external tools, opening apps, and taking steps to deliver final results (PPTs, Reports, Data Pipelines).
5. **Scheduled Workflow Execution**: Describe a task once, and it runs periodically or on a specific schedule without re-prompting.

## Workflow Logic
- **Stage 1 (VM Provisioning)**: Agent spawns or connects to a persistent execution environment (simulated via background processes).
- **Stage 2 (Pattern Recognition)**: Matches user intent against the "Collective Knowledge Base" of proven business workflows.
- **Stage 3 (Autonomous Execution)**: The agent works through a multi-step checklist, interacting with required tools (web, APIs, docs).
- **Stage 4 (Proactive Delivery)**: Delivers results and alerts the user upon completion or trigger event.
