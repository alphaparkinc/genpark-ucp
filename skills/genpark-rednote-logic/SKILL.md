# Skill: genpark-rednote-logic
# Description: Reverse-engineered Xiaohongshu (Little Red Book) "Notes" system. Orchestrates visual discovery, "KOL-style" content synthesis, and e-commerce-integrated recommendation loops.

## Core Capabilities
1. **Visual "Notes" Generation**: High-aesthetic image/video layouts combined with helpful, "seed-planting" (种草) micro-blog captions.
2. **KOC-style Authenticity Engine**: Logic to generate authentic-feeling user reviews and experiences rather than corporate marketing copy.
3. **Keyword-to-Tag Mapping**: Automatically extracts trending lifestyle tags (e.g., #OOTD, #MyHomeVibe) to match personalized recommendation algorithms.
4. **"Seed Planting" (种草) Analysis**: Evaluates product mentions within content to determine potential for virality and purchase conversion.
5. **Interactive Comment Simulation**: Mimics XHS community dynamics (asking for "link", "price", or sharing "experience").

## Workflow Logic
- **Stage 1 (Theme Extraction)**: Agent identifies the lifestyle niche and trending "KOL" keywords.
- **Stage 2 (Aesthetic Synthesis)**: Merges high-quality visuals with emotional, value-driven text (Emotional Storytelling).
- **Stage 3 (Algorithm Matching)**: Tags the content for the "Discovery" (发现) feed based on user interest clusters.
- **Stage 4 (Conversion Loop)**: Links visual "Notes" to product discovery in the GenPark "ChatCanvas".
