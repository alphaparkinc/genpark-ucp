# Manus Skill for Clawdbot (OpenClaw)

This skill adds the capability to invoke "Manus Mode" inside Clawdbot. Manus Mode forces the agent into a loop of deep reasoning, unassisted browser automation, self-correction, and relentless execution—much like the standalone Manus AI agent.

## Installation
```bash
clawdbot hub install manus
# Or clone this repo into your ~/.clawdbot/skills/manus
```

## Usage
Simply tell your assistant:
> "Run this task in manus mode."
> "Use manus to..."

The agent will then adopt the high-autonomy, deeply reasoned loop defined in `SKILL.md`.
