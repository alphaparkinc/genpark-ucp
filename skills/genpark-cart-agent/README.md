# GenPark Cart Agent

## 🌟 What is GenPark Cart?
GenPark Cart is an **"Actionable Shopping Decision Cart"** generated by AI. 
Unlike traditional e-commerce where users search for individual items (SKUs) and add them to a static cart, GenPark Cart operates on **Intent + Scenarios**. 

You tell the agent: *"I need a Summer Clean Girl Outfit under $120"* or *"Work From Home Setup"*, and the Agent builds a **cohesive bundle** (a Cart) that fits the style, budget, and inventory constraints in real-time.

## 🚀 Key Features Emulated by this Skill
- **Dynamic Bundle Generation**: Connects items together conceptually (e.g., Top + Skirt + Shoes + Bag).
- **Constraint Engine (Solver)**: Forces the total price ≤ budget, ensures style consistency.
- **Real-Time Commerce Layer**: Fetches live prices and availability (Shopify, Amazon, TikTok Shop simulation).
- **Actionable Execution (V1)**: Provides direct checkout or product links for the entire bundle.
- **Cart Operations**: Users can ask to `[Swap]` an item, `[Change Budget]`, or `[Change Style]`.
