# Skill: genpark-lemon-lifestyle
# Description: Reverse-engineered Lemon8-style lifestyle sharing engine. Orchestrates visual storytelling, aesthetic curation, and interest-based community engagement (For You Feed).

## Core Capabilities
1. **Aesthetic Post Generation**: Automates the creation of "Lemon8-style" posts: high-quality visuals combined with informative, blog-like captions and overlays.
2. **Interest-based Curation (Discovery Engine)**: Mimics the "For You" algorithm by categorizing content into lifestyle themes (Fashion, Beauty, Food, Travel, Home).
3. **Smart Template System**: Provides presets for "Outfit of the Day", "Healthy Recipes", and "Travel Guides" with standardized visual layouts.
4. **Community Engagement Simulation**: Automates "Lemon8-style" interactions (authentic-feeling comments, appreciation, and save/share patterns).
5. **SEO-optimized Tagging**: Recommends viral hashtags and categories based on real-time lifestyle trends.

## Workflow Logic
- **Stage 1 (Theme Selection)**: Agent identifies the lifestyle niche (e.g., "Minimalist Home").
- **Stage 2 (Visual Curation)**: Fetches or generates aesthetic imagery using DALL-E/Midjourney-style prompts.
- **Stage 3 (Copywriting)**: Drafts a "Micro-blog" style caption with structured advice (bullet points, emojis, helpful tips).
- **Stage 4 (Distribution)**: Maps the post to the "Knowledge Garden" for discovery by other agents/users.
